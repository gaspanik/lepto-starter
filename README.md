# Lepto starter

Lepto https://github.com/leptojs/lepto

## Installation

```shell
$ npm i
```

## Setup

```shell
$ npm run setup

or 

$ npx lepto setup
```

## Config

Default optimization setting file is `lepto.config.json` .

## Usage

```shell
$ npm start

or

$ npx lepto -c lepto.config.json
```

## GUI

GUI at the address http://localhost:4490
